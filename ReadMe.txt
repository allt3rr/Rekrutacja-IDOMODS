Opis projektu rekrutacyjnego dla ISOMODS

Projekt został przygotowany na potrzeby procesu rekrutacyjnego do firmy ISOMODS. Został zbudowany przy użyciu HTML, CSS oraz JavaScript.

W projekcie wykorzystano bibliotekę Swiper.js, która odpowiada za implementację slidera.

Dla usprawnienia procesu kodowania wykorzystałem kod komponentu custom select ze strony W3Schools, który następnie został odpowiednio zmodyfikowany i dostosowany do potrzeb projektu.

Całość została zaprojektowana z myślą o SEO – struktura kodu, grafiki oraz pozostałe elementy zostały zoptymalizowane zgodnie z wytycznymi.
